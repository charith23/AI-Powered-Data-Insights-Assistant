# 🚀 AI-Powered Data Insights Assistant

An AI-powered full-stack application that allows users to query a SQL database using natural language and instantly get results.

---

## 🧠 Problem Statement

In many organizations:

- Non-technical users cannot write SQL queries  
- Developers spend time writing and debugging repetitive queries  
- Data exists, but insights are slow to extract  
- Business teams depend on developers for simple data requests  

👉 Accessing data is slow, technical, and inefficient  

---

## 💡 Solution

This project solves the problem by enabling:

- Natural language → SQL conversion using AI  
- Automatic query execution on a PostgreSQL database  
- Instant results displayed in a chat-based UI  

👉 Users can simply type:

**“Show total sales by region”**

And the system:

1. Converts it to SQL  
2. Executes it  
3. Returns results instantly  

---

## ⚙️ Tech Stack

- **Frontend:** React.js  
- **Backend:** Node.js + Express  
- **Database:** PostgreSQL  
- **AI Model:** Ollama (tinyllama - local LLM)  

---

## 🔥 Key Features

- Natural Language to SQL conversion  
- Chat-based user interface  
- Real-time database querying  
- Local AI integration (no API cost)  
- SQL safety filtering (only SELECT queries allowed)
